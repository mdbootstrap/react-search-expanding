# MDB React 5

Responsive React Expanding Search Bar built with Bootstrap 5. Examples with the trigger on click and on hover.

Check out [React Expanding Search Bar Documentation](https://mdbootstrap.com/docs/react/extended/search-expanding/) for detailed instructions & even more examples.

## Examples

[On click:
![Expanding Search Bar #1](https://user-images.githubusercontent.com/108793661/186087072-82b09629-aac3-40cc-b3e6-6c24004baa1c.png)](https://mdbootstrap.com/docs/react/extended/search-expanding#section-on-click)

[On Hover:
![Expanding Search Bar #2](https://user-images.githubusercontent.com/108793661/186087086-79429272-a654-4159-8c4e-b92a8de72396.png)](https://mdbootstrap.com/docs/react/extended/search-expanding#section-on-hover)

## How to use?

1. Download MDB 5 - FREE REACT UI KIT

2. Choose your favourite customized component and click on the image

3. Copy & paste the code into your MDB project
